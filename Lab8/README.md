### Завдання 2 – Власний функціональний інтерфейс
- [x] Створено інтерфейс `Printable` з методом `void print()`
- [x] Реалізовано лямбда вираз для цього інтерфейсу

### Завдання 3a – Predicate
- [x] Лямбда для перевірки, чи рядок можна привести до числа
- [x] Використання методу `and()` для комбінації умов

### Завдання 3b – Consumer
- [x] Лямбда для виведення повідомлень про початок та кінець пари
- [x] Використання методу `andThen()`

### Завдання 3c – Supplier
- [x] Лямбда для генерації трьох випадкових великих літер

### Завдання 3d – Function
- [x] Лямбда для обчислення добутку чисел з рядка

### Завдання 4 – Stream API
- [x] Створення стріму з масиву `Product`
- [x] Отримання всіх брендів (`map`, `distinct`)
- [x] Отримання 2 товарів з ціною менше 1000 (`filter`, `limit`)
- [x] Сума всіх товарів на складі (`mapToInt`, `sum`)
- [x] Групування товарів за брендом (`Collectors.groupingBy`)
- [x] Сортування товарів за ціною (`sorted`, `toArray`)

### Завдання 5 – Посилання на методи
- [x] Заміна лямбд на посилання на методи (`Product::getBrand`, `Product[]::new`)

### Завдання 6 – Optional
- [x] Пошук максимального значення в масиві чисел
- [x] Обробка порожнього списку з поверненням рядка "Числа відсутні"
- [x] Використання методів `orElseThrow()`, `map()`, `orElse()`
