## ✅ Виконані завдання

### Завдання 2 – Клас, що розширює Thread
- [x] Створено клас `MyThread`, який виводить 100 разів "Я люблю програмувати!!!"
- [x] Виведено ім'я, пріоритет, стан (NEW, RUNNABLE, TERMINATED)
- [x] Змінено ім'я та пріоритет потоку
- [x] Використано `join()` для очікування завершення

### Завдання 3 – Клас, що реалізує Runnable
- [x] Виведення чисел від 0 до 10000, що діляться на 10
- [x] Обробка переривання потоку (`isInterrupted()`)
- [x] 3 потоки, `Thread.sleep()` та `interrupt()`

### Завдання 4 – Арифметична прогресія (синхронізований метод)
- [x] Виведення арифметичної прогресії від 1 до 100 з кроком 1
- [x] Використання `synchronized` методу
- [x] 3 потоки

### Завдання 5 – Блок синхронізації
- [x] Перероблено завдання 4 з використанням блоку `synchronized`

### Завдання 6 – Reader та Printer (wait/notify)
- [x] Потік Reader читає рядок з консолі
- [x] Потік Printer виводить рядок
- [x] Використання `wait()` та `notify()`
- [x] Вихід за словом `exit`

### Завдання 7 – Сума масиву (ExecutorService)
- [x] Масив на 1 000 000 випадкових чисел
- [x] Однопотокове обчислення суми
- [x] Багатопотокове обчислення (ExecutorService, 5 потоків, Callable)
- [x] Вимірювання часу виконання
