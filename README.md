# Java_labs_ztu

Лабораторні роботи з Java

## Lab1 – Знайомство з мовою програмування Java

### Виконані завдання:
- Task1: Виведення "Hello, World!!!"
- Task2: Додавання двох цілих чисел
- Task3: Виведення аргументів командного рядка
- Task4: Найбільший спільний дільник (НСД)
- Task5: Сума цифр числа
- Task6: Послідовність Фібоначчі (пряма та зворотна)
- Task7: Виведення ASCII кодів літер A-Z

### Структура:

src/
└── com/education/ztu/
├── Task1.java
├── Task2.java
...
└── Task7.java


## Як запустити:
1. Відкрити проєкт в IntelliJ IDEA
2. Запустити будь-який TaskX.java
3. Для Task3 передати аргументи: `2 3 5 8`
